swarm CLI command reference

A place to quickly look up help the reference for swarm commands.

Command Description
swarm infoGet basic information on context and environment
swarm upDefine and start a new app
swarm createDefine a new app by loading a json file
swarm startStart an application or service
swarm stopStop an application or service
swarm statusShow current status of an app or service
swarm updateUpdate a component
swarm scaleupIncrease number of instances running a component
swarm scaledownReduce number of instances running a component
swarm logsPrint an app's standard output and error messages
swarm statsPrint basic statistics of an instance
swarm lsList all apps
swarm envHandle environments
swarm catShow the configuration of an application
swarm userCreate and modify users
swarm companyManage companies
swarm deleteDelete an app. Note that you lose all contained data by doing so.
swarm versionPrint swarm client version (Reference not yet available)
swarm loginLog in to your account (Reference not yet available)
swarm logoutLog out from your account (Reference not yet available)
swarm completionSetup CLI completion (Reference not yet available)
