Quick Start

Your easy way to get going with your new Giant Swarm account

  • Mac + Homebrew
  • Linux
  • Installation instructions for all platformsOther platforms 
$ brew tap giantswarm/swarm
$ brew install swarm-client
$ git clone https://github.com/giantswarm/helloworld.git
$ cd helloworld
$ swarm login yourusername

Now enter your Giant Swarm credentials.

$ swarm up --var=domain=helloworld-yourusername.gigantic.io

Then open your running application at helloworld-yourusername.gigantic.io
